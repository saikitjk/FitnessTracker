# FitnessTracker

## Description

A full-stack workout tracker application with HTML, CSS, JavaScript, jQuery, Ajax, APIs, Node.js, MySQL, Node, Express, Handlebars and ORM.

Application has been deployed through Heroku.

## Deployed application
